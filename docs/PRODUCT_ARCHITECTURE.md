# ReachScan public product architecture

| Product | Public role | Recommended home |
|---|---|---|
| Core Research Article v1.0-RC8 | Primary concise scientific claim | arXiv; venue submission |
| Supplementary Evidence Appendix v1.0-RC8 | Reviewer-facing controls and supporting analyses | arXiv attachment / submission supplement |
| Full Technical Report and Audit Record v0.9.34 | Complete scientific, methodological, and audit record | Zenodo report DOI |
| Reach-Scan Evidence and Reproducibility Archive v1.0-RC2 | Canonical data, constructors, registries, provenance, and publication snapshots | Zenodo dataset DOI |
| `reachscan` v0.2.5 | Installable measurement core for reuse and extension | GitHub + Zenodo software DOI |
| Historical Lineage Source Capsules v1.0-RC1 | Optional forensic development history | Zenodo secondary record or restricted archival storage |

The paper is the citation entry point. The report carries the complete record. The archive supports verification. The software supports reuse. The optional lineage volumes are not required for current-paper reproduction.
